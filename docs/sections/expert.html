Prerequisites Required

This track assumes solid Python skills, understanding of ML concepts (training, evaluation, overfitting), familiarity with LLMs, embeddings, and RAG. You should have completed both the Beginner and Intermediate tracks, or have equivalent professional experience.

Beginner Track Intermediate Track

What You'll Learn

Advanced Prompting

Constitutional AI, structured outputs, meta-prompting, and prompt injection defense. The techniques professional AI engineers use daily.

AI Agents

Build autonomous agents with tool use, planning loops, memory systems, and multi-agent orchestration using LangGraph and AutoGen.

LLM Evaluation

Systematic evaluation frameworks: evals, red-teaming, RAGAS for RAG systems, and building automated test suites for LLM outputs.

Fine-Tuning LLMs

When and how to fine-tune. LoRA, QLoRA, instruction tuning, and RLHF. Know when fine-tuning beats prompt engineering.

Responsible AI

Bias, fairness, transparency, and safety. Build AI systems that are trustworthy, explainable, and aligned with human values.

Multimodal AI

Vision-language models, image generation, audio processing. Build apps that see, hear, and speak — beyond text.

Production Deployment

Containerization, rate limiting, cost optimization, monitoring, and scaling AI services from prototype to production.

Capstone Project

Apply everything you've learned. Build a complete, production-ready AI system from requirements to deployed application.

Anatomy of an AI App

Every AI application — no matter how complex — is built from the same 4 components. Once you understand these, you can read and build any AI app.

⌨️

Input

What the user provides. Text, files, images, voice, form data.

📋

Prompt

How you package the input + context + instructions for the model.

🤖

Model

The LLM that processes the prompt and generates a response.

📤

Output

The model's response — displayed, stored, or used to trigger another action.

🔄

Conversation = Repeated Loop

A chatbot is just this 4-step loop running continuously. Each turn, the full conversation history is included in the prompt so the model has context. This is called the "context window".

Minimal Working Chatbot

Here is the complete minimal chatbot — ~30 lines of Python. Read every line carefully.

import anthropic

client = anthropic.Anthropic(api_key="your-key")

SYSTEM_PROMPT = """You are a helpful AI study assistant for students learning AI concepts.
Your goals:
- Explain AI concepts clearly with analogies and examples
- Ask follow-up questions to check understanding
- When a student seems confused, try a different explanation
- Keep responses concise (under 150 words unless asked for more)"""

conversation_history = []

print("🤖 AI Study Assistant ready! (type 'quit' to exit)\n")

while True:
    user_input = input("You: ").strip()

    if user_input.lower() in ["quit", "exit", "q"]:
        print("Goodbye! Keep learning! 🚀")
        break

    if not user_input:
        continue

    # Add user message to history
    conversation_history.append({
        "role": "user",
        "content": user_input
    })

    # Call the model with full history
    response = client.messages.create(
        model="claude-3-5-sonnet-20241022",
        max_tokens=1024,
        system=SYSTEM_PROMPT,
        messages=conversation_history
    )

    assistant_message = response.content[0].text

    # Add assistant response to history (important!)
    conversation_history.append({
        "role": "assistant",
        "content": assistant_message
    })

    print(f"\n🤖 Assistant: {assistant_message}\n")

What Each Part Does

Code ElementPurposeWhat happens if you remove it
SYSTEM_PROMPTGives the assistant its persona and rulesModel defaults to generic helpful assistant
conversation_historyStores all messages so model has contextModel forgets previous messages — goldfish memory
messages=conversation_historySends the full history on every API callSame as above — each call would be independent
Appending assistant reply to historyModel can reference its own previous responsesModel can't say "as I mentioned" — loses coherence

Making It Better: 4 Easy Upgrades

1. Add Streaming (Responses Appear Word by Word)

with client.messages.stream(
    model="claude-3-5-sonnet-20241022",
    max_tokens=1024,
    system=SYSTEM_PROMPT,
    messages=conversation_history
) as stream:
    full_response = ""
    print("\n🤖 Assistant: ", end="", flush=True)
    for text in stream.text_stream:
        print(text, end="", flush=True)
        full_response += text
    print("\n")

2. Add Context Window Management

LLMs have a maximum context length. If the conversation gets too long, trim old messages.

MAX_HISTORY = 20  # Keep last 20 messages

# After appending the assistant response:
if len(conversation_history) > MAX_HISTORY:
    # Keep first message (often important) + last MAX_HISTORY-1
    conversation_history = conversation_history[-MAX_HISTORY:]

3. Save & Load Conversations

import json, os

HISTORY_FILE = "chat_history.json"

def load_history():
    if os.path.exists(HISTORY_FILE):
        with open(HISTORY_FILE) as f:
            return json.load(f)
    return []

def save_history(history):
    with open(HISTORY_FILE, "w") as f:
        json.dump(history, f, indent=2)

conversation_history = load_history()
# ... at the end of each turn:
save_history(conversation_history)

4. Topic-Specific Study Mode

topics = {
    "1": ("Machine Learning Basics", "Focus on ML fundamentals: training, inference, features, labels. Quiz the student frequently."),
    "2": ("Neural Networks", "Explain neural networks with layer-by-layer analogies. Use the neuron → brain analogy."),
    "3": ("LLMs & Transformers", "Deep dive into transformers: attention, tokens, embeddings. Use the 'lookup table' analogy for embeddings."),
}

print("Choose a study topic:")
for key, (name, _) in topics.items():
    print(f"  {key}. {name}")

choice = input("Enter number: ").strip()
topic_name, topic_system = topics.get(choice, ("General AI", "Discuss general AI topics."))
SYSTEM_PROMPT = f"You are an AI tutor specializing in {topic_name}. {topic_system}"

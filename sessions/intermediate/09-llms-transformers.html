LLMs and Transformers

How does ChatGPT actually work? Tokens, embeddings, attention, context windows, and why LLMs hallucinate — explained without the math fear.

75 min Intermediate Interactive Lab 5 Quiz Questions

Tokens: The Currency of LLMs

LLMs don't process words — they process tokens. A token is roughly a word or word-piece (subword). The model converts text into token IDs, processes them, then converts token IDs back to text.

💡

Token Rules of Thumb

~1 token per 4 characters of English text. "Hello world" ≈ 2 tokens. A 5-page essay ≈ ~2,000 tokens. GPT-4's 128k context window = ~96,000 words. Why tokens matter: you pay per token and your prompt + response must fit in the context window.

import tiktoken  # pip install tiktoken

# OpenAI's tokenizer (same as GPT-4)
enc = tiktoken.encoding_for_model("gpt-4")

text = "Hello! I'm learning about transformers and LLMs."
tokens = enc.encode(text)
print(f"Text: '{text}'")
print(f"Token IDs: {tokens}")
print(f"Token count: {len(tokens)}")

# Decode individual tokens to see the subwords
for token_id in tokens:
    decoded = enc.decode([token_id])
    print(f"  {token_id} → '{decoded}'")

Embeddings: Meaning as Numbers

Each token is converted to a high-dimensional vector (the embedding). The magic: similar words have similar embeddings. "King" and "Queen" are close in embedding space. "Cat" and "automobile" are far apart.

GPT-4 uses 12,288-dimensional embeddings — each token is a point in 12,288-dimensional space. Relationships between concepts are encoded as geometric distances and directions in this space.

✨

The Famous Example

embedding("King") - embedding("Man") + embedding("Woman") ≈ embedding("Queen"). The geometric relationship between gender is encoded as a consistent direction in embedding space. This is how LLMs "understand" analogies.

Attention: The Core of Transformers

The key innovation in the 2017 "Attention Is All You Need" paper. Attention lets each token look at every other token in the sequence to decide what to focus on.

Analogy: Reading the sentence "The animal didn't cross the street because it was too tired." What does "it" refer to? "Animal" or "street"? Your brain uses attention — looking back at context to resolve the ambiguity. Transformers do the same thing mathematically.

Self-Attention Intuition For each token, attention computes:
Query (Q): "What am I looking for?"
Key (K): "What do I advertise about myself?"
Value (V): "What information do I carry?"

Each token's query is matched against all other tokens' keys. High match → high attention weight → more of that token's value is incorporated into the current token's representation.

Why LLMs Hallucinate

LLMs generate the statistically most likely next token given the context. They are not knowledge databases. They have no way to say "I don't know this" — they will always generate something plausible-sounding.

🎲

Statistical Generation

The model picks the next token based on probability distributions — not factual lookup. Plausible ≠ true.

📚

Training Data Gaps

If a fact wasn't well-represented in training data, the model learned a wrong or uncertain distribution for it.

📏

Context Window Limit

The model forgets information beyond its context window — it can't reference documents it hasn't been given.

🔇

Fix: RAG

Retrieval-Augmented Generation (Session 11) grounds the model in actual documents, dramatically reducing hallucination.

# Experiment: See tokens and context limits in action
import anthropic

client = anthropic.Anthropic()

# Token counting
response = client.messages.count_tokens(
    model="claude-3-5-sonnet-20241022",
    messages=[{"role": "user", "content": "Explain transformers in detail..."}]
)
print(f"Input tokens: {response.input_tokens}")

# Context window experiment: give the model a document and ask about it
document = "The revenue for Q3 was $4.2M. The CEO is Sarah Chen."
question = "What was the Q3 revenue?"

# With document (correct answer expected)
response_with_context = client.messages.create(
    model="claude-3-5-sonnet-20241022",
    max_tokens=100,
    messages=[{
        "role": "user",
        "content": f"Document: {document}\n\nQuestion: {question}"
    }]
)
print("With context:", response_with_context.content[0].text)

# Without document (hallucination risk!)
response_without = client.messages.create(
    model="claude-3-5-sonnet-20241022",
    max_tokens=100,
    messages=[{"role": "user", "content": question}]
)
print("Without context:", response_without.content[0].text)
